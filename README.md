# 夸父 (Kuafu)

夸父 — 自我进化的 AI Agent。

**永不停止地追逐目标，每一次执行都是进化的一步。**

夸父不是一个被动的工具，它是一个活的 agent。每次任务完成后，它自动反思、学习、优化自己的能力。用户感觉不到进化的过程，只知道夸父越来越好用。

## 核心理念

- **进化 = 工作的自然产物**，不是额外操作
- **核心不可破坏**，core/ 目录只读保护区
- **身份感知**：知道自己是谁，用户是谁，边界在哪里
- **向后兼容**：V1 接口永久不变
- **先能用再进化**：V1 能干活，V2 开始自我改进

## 架构

```
kuafu/
├── core/          ← 只读保护区（身份/沙盒/记忆/进化引擎）
├── strategy/      ← 可进化区（prompt 模板/任务策略）
├── skills/        ← 可进化区（可复用技能包）
└── memory/        ← 记忆数据（Hindsight 后端）
```
